# Dunbar Link 베타 테스트 가이드

## 1. 테스트 목적

- 던바링크는 가까운 사람을 정리하고, 신호를 보내고, 연결 상태를 확인하는 앱이에요.
- 이번 테스트는 디자인 평가보다는 **기본 동작이 잘 되는지** 확인하는 게 목적이에요.
- 편하게 5분 정도만 써보고, 이상한 부분이 있으면 알려주시면 돼요.

## 2. 테스트 전 준비

- 휴대폰의 **크롬(안드로이드)** 또는 **사파리(아이폰)** 에서 링크로 접속해 주세요.
- 가능하면 **홈 화면에 추가**해서 앱처럼 실행하면 더 좋아요.
- 앱이 이상하면 **새로고침**, 그래도 이상하면 **앱을 완전히 종료한 뒤 다시 실행**해 주세요.

## 3. 테스트 순서

1. 링크로 앱에 접속해요.
2. **Me 탭**에서 내 이름을 입력해요.
3. 프로필 **사진**을 추가해요.
4. **저장하기**를 눌러요.
5. **Home 탭**을 확인해요.
6. (초대를 받았다면) **초대 링크**를 열어 연결을 수락해요.
7. 연결된 사람이 **Home / People**에 보이는지 확인해요.
8. 연결된 사람에게 **신호 보내기**를 해봐요.
9. 받은 쪽에서 **파란 점**(타일 왼쪽 위)과 위쪽 **말풍선 빨간 점**이 보이는지 확인해요.
10. 신호를 **읽으면** 그 점들이 사라지는지 확인해요.
11. 내가 바꾼 **이름/사진**이 상대 화면에도 보이는지 확인해요.
12. 사람을 **폴더**에 넣어보고, 다시 꺼내봐요.
13. Home 아래쪽 **추천 보기**를 눌러봐요.

## 4. 확인해 주세요

- [ ] 이름이 잘 저장되나요?
- [ ] 사진이 잘 올라가나요?
- [ ] 상대방에게 내 이름과 사진이 보이나요?
- [ ] 초대 수락이 잘 되나요?
- [ ] 신호를 보내고 받을 수 있나요?
- [ ] 파란 점이 보이고, 읽으면 사라지나요?
- [ ] Home / People / Me 탭 이동이 자연스러운가요?
- [ ] 폴더에 사람을 넣고 뺄 수 있나요? (폴더 **가운데**에 끌어다 놓으면 들어가요)
- [ ] 화면이 너무 답답하거나 이상한 부분은 없나요?

## 5. 문제가 생기면 보내주세요

이렇게 알려주시면 큰 도움이 돼요.

- 어떤 휴대폰인지 (예: 아이폰 14, 갤럭시 S23)
- 어떤 브라우저인지 (사파리 / 크롬)
- 어느 화면에서 문제가 생겼는지
- 가능하면 **스크린샷**
- 어떤 순서로 눌렀는지
